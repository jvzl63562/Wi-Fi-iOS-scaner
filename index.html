Guia de Uso

1 Como ver redes Wi-Fi disponíveis

No iOS, você pode ver as redes Wi-Fi disponíveis através das Configurações:

  1. Abra o aplicativo "Configurações"
  2. Toque em "Wi-Fi"
  3. Certifique-se de que o Wi-Fi esteja ativado
  4. As redes disponíveis aparecerão na lista "REDES"
2 Como ver a senha de uma rede Wi-Fi salva (iOS 16+)

A partir do iOS 16, você pode ver senhas de redes Wi-Fi salvas:

  1. Abra o aplicativo "Configurações"
  2. Toque em "Wi-Fi"
  3. Toque no ícone de informação (i) ao lado da rede conectada
  4. Toque em "Senha"
  5. Use Face ID, Touch ID ou digite sua senha para visualizar
3 Como compartilhar uma rede Wi-Fi (iOS 11+)

Você pode compartilhar facilmente uma rede Wi-Fi com outros dispositivos Apple:

  1. Certifique-se de que ambos os dispositivos estejam desbloqueados
  2. O dispositivo receptor deve tentar se conectar à rede Wi-Fi
  3. Aproxime os dispositivos
  4. No seu iPhone, aparecerá uma notificação "Compartilhar Senha Wi-Fi"
  5. Toque em "Compartilhar" para enviar a senha
4 Como gerar um QR Code para sua rede Wi-Fi (iOS 18+)

No iOS 18, você pode gerar QR Codes para suas redes Wi-Fi:

  1. Abra o aplicativo "Senhas"
  2. Toque na opção "Wi-Fi"
  3. Selecione a rede desejada
  4. Toque em "Mostrar QR Code da Rede"
  5. O QR Code será exibido e pode ser compartilhado

Alternativamente, você pode usar a aba "QR Code" neste aplicativo para gerar códigos para qualquer rede.

Limitações no iOS

Devido às restrições de segurança do iOS, aplicações web não podem:

  • Acessar diretamente informações sobre redes Wi-Fi disponíveis
  • Recuperar senhas de redes salvas automaticamente
  • Conectar-se automaticamente a redes Wi-Fi
  • Realizar ataques de força bruta para descobrir senhas

Estas limitações são medidas de segurança implementadas pela Apple para proteger a privacidade dos usuários.

Gerador de QR Code para Wi-Fi

Crie QR Codes para compartilhar facilmente suas redes Wi-Fi com outros dispositivos.

Escaneie este QR Code com a câmera de outro dispositivo para conectar-se à rede Wi-Fi.

Como usar QR Codes de Wi-Fi

1 Escaneando um QR Code

Para escanear um QR Code de Wi-Fi:

  1. Abra o aplicativo "Câmera" no seu iPhone
  2. Aponte a câmera para o QR Code
  3. Uma notificação aparecerá
  4. Toque na notificação para conectar-se à rede
2 Compartilhando o QR Code

Você pode compartilhar o QR Code gerado:

  1. Faça uma captura de tela do QR Code
  2. Compartilhe a imagem por mensagem, e-mail ou AirDrop
  3. O destinatário pode escanear o código com a câmera
